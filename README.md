# PYAWM

Python-библиотека для математического моделирования оптических явлений в волноведущих структурах асимптотическими методами.

## Структура проекта

```
PYAWM/
├── pyawm/                      # Основной пакет с исходным кодом
│   ├── __init__.py             # Инициализация пакета
│   ├── pyawm.py                # Ядро: асимптотические методы, уравнения Максвелла
│   ├── 2d_waveguide.py          # Модель 3-слойного плавно-нерегулярного волновода
│   ├── luneberg_lense.py        # Модель 4-слойной градиентной линзы Luneberg
│   ├── solveHomoSLAE.py         # Решатель однородных систем линейных уравнений
│   ├── preview_wrappers.py      # Утилиты рендеринга уравнений в LaTeX
│   ├── IndexProf_2000_points.txt  # Данные индекса преломления линзы
│   └── luneberg_lens_profile.txt  # Выходные профили линзы
├── notebooks/                  # Jupyter-тетрадки для исследований
│   ├── lens_profile.ipynb      # Вычисление профиля линзы Luneberg
│   ├── mmcp24.ipynb            # Материалы конференции MMCP 2024
│   ├── determinant compute speed-up.ipynb  # Оптимизация вычисления детерминанта
│   └── luneberg_lens_numbers.txt  # Численные данные линзы
├── images/                     # Генерируемые изображения (по датам)
└── requirements.txt            # Зависимости Python
```

## Описание файлов

### Модули ядра (`pyawm/`)

**`pyawm.py`** — основной модуль с реализацией асимптотических методов:
- `gen_vars_subs()` —  генерация подстановок переменных
- `gen_subs_diff()` —  генерация дифференциальных подстановок
- `gen_maxwell_eqs()` —  построение уравнений Максвелла с асимптотическим разложением
- `layered_sols()` —  построение решений для слоёв волновода
- `gen_boundry_conds()` —  генерация граничных условий на границах слоёв
- `boundry_check_all_solutions()` —  проверка корректности решений

**`2d_waveguide.py`** —  моделирование плавно-нерегулярного трехслойного волновода (cover/film/substrate):

**`luneberg_lense.py`** — модель четырёхслойной градиентной линзы Luneberg:

**`solveHomoSLAE.py`** —  алгоритм решения однородных систем линейных алгебраических уравнений:
- `solve_subsystem()` —  решение подсистемы 2×3
- `solve_all()` —  поиск всех ветвей решений
- `subsystems_permutations()` —  перебор вариантов пар уравнений

**`preview_wrappers.py`** —  утилиты для рендеринга LaTeX-уравнений в изображения

### Jupyter-тетрадки (`notebooks/`)

| Файл | Описание |
|------|----------|
| `lens_profile.ipynb` | Вычисления профиля линзы Luneberg |
| `mmcp24.ipynb` | Презентация для конференции MMCP 2024 |
| `determinant compute speed-up.ipynb` | Оптимизация скорости вычисления детерминанта |
| `luneberg_lens_numbers.txt` | Численные данные для линзы |

### Данные (`pyawm/`)

- `IndexProf_2000_points.txt` — профиль индекса преломления (2000 точек)
- `luneberg_lens_profile.txt` — вычисленный профиль линзы

### Изображения (`images/`)

В директории `images/` сохраняются генерируемые программами графики и визуализации. Структура организована по датам (например, `2024-09-02/`, `2024-09-03/` и т.д.).

## Установка

1. Создайте виртуальное окружение:
```bash
python -m venv .venv
source .venv/bin/activate
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

### Jupyter Notebook

```bash
jupyter-notebook
```

Затем откройте нужную тетрадку из директории `notebooks/`.

### Модули Python

Импорт модулей:
```python
from pyawm.pyawm import *
from pyawm import pyawm, preview_wrappers, solveHomoSLAE
```

Запуск модулей напрямую:
```bash
python pyawm/2d_waveguide.py
python pyawm/luneberg_lense.py
```

## Зависимости

- **sympy** — символьные вычисления
- **numpy, scipy** — численные расчёты и поиск корней
- **matplotlib** — визуализация
- **f2py** (через sympy) — Fortran-бэкенд для быстрого вычисления детерминантов
